# Opportunity Inbox Copilot

AI-powered system for parsing student emails and ranking opportunities based on relevance, urgency, and personal profile.

## Overview

Students receive numerous emails about scholarships, internships, competitions, and fellowships, but many go unnoticed or ignored. This project builds an intelligent system that:

- Detects real opportunities from emails  
- Extracts key details (deadlines, eligibility, requirements)  
- Matches them with a student's profile  
- Ranks them by priority with clear reasoning  

This project is based on the SOFTEC 2026 AI Hackathon problem statement.

---

## Features

- Email Parsing & Classification  
  Identify whether an email contains a real opportunity  

- Information Extraction  
  Extract:
  - Opportunity type  
  - Deadlines  
  - Eligibility criteria  
  - Required documents  
  - Contact/application links  

- Student Profile Matching  
  Match opportunities with:
  - Degree/program  
  - Semester  
  - CGPA  
  - Skills & interests  
  - Preferred opportunity types  
  - Location preference  

- Smart Ranking System  
  Rank opportunities based on:
  - Relevance to profile  
  - Urgency (deadline proximity)  
  - Completeness  


##  Tech Stack (Suggested)

- Frontend: React / Next.js  
- Backend: Node.js / FastAPI  
- AI/NLP: OpenAI API / spaCy / Transformers  
- Database: MongoDB / PostgreSQL  
- Parsing: Regex + NLP pipelines  


## Demo Flow

1. Upload or paste emails  
2. Enter student profile  
3. Run analysis  
4. View:
   - Detected opportunities  
   - Extracted details

   - Action steps
